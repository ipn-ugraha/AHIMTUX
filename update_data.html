⚡ Update Data AHI MTU

Upload Excel → Generate JSON → Push ke GitHub

1 Konfigurasi GitHub
Belum terhubung
Buat di: GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
Centang permission: repo (full control)
Contoh: ahi_data.json atau data/ahi_data.json
❓ Cara membuat Personal Access Token (PAT)
  1. Buka github.com/settings/tokens
  2. Klik Generate new token (classic)
  3. Beri nama, misal: AHI MTU Dashboard
  4. Set Expiration sesuai kebutuhan (90 days / No expiration)
  5. Centang permission: repo (full control of private repositories)
  6. Klik Generate token — salin token, simpan di tempat aman
  7. Token hanya tampil sekali — jika hilang, buat baru
⚠ Jangan bagikan token ke orang lain. Setiap anggota tim sebaiknya membuat token sendiri.
🔒 Token hanya dikirim langsung ke GitHub API — tidak melewati server lain.
⚠ Di PC bersama: isi token tiap kali, jangan centang Simpan token.
